# rook2root.co

## Getting Started

run the development server:

```bash
npm run dev
```

## Deploy

Cloudflare automatically deploys every push to the main branch.
In order to deploy develop from the main branch:

```bash
git merge develop
git push
```

## Tips

serve staticly built project:

```bash
npx serve out
```
