---
title: Urgency
description: Interface elements that suggest limited time for decision-making, pressuring users to act quickly. These signals are often arbitrary or unverifiable, aimed at reducing user deliberation.
---

## Countdown Timer
A visual timer is displayed to imply that an offer or availability window is about to expire.

## Limited-time Message
Users are informed that a deal is ending soon, without specifying an exact deadline or with vague timing.
